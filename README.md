# LoRa Workshop

## RN2483

abp setup

```
mac set devaddr <device address>
mac set nwkskey <network session key>
mac set appskey <application session key>
mac save
```

otaa setup

```
mac set appeui <appEUI>
mac set appkey <app key>
mac save
```
